# CodeAlpha Web Scraping Task

This project is submitted for the CodeAlpha Data Analytics Internship.

## Task Details:
- Task: Web Scraping
- Website Scraped: http://quotes.toscrape.com
- Tool Used: Python, BeautifulSoup

## Features:
- Extracts quotes and authors
- Saves data in CSV format

## How to Run:
1. Install required libraries:
   pip install requests beautifulsoup4 pandas
2. Run the script:
   python web_scraping.py

## Output:
- quotes_data.csv (contains scraped data)
